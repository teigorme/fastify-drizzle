## 📦 Instalação

```bash
# Clonar o repositório
git clone https://github.com/teigorme/fastify-drizzle.git

# Entrar no diretório
cd fastify-drizzle

# Instalar dependências
pnpm install

# Rodar o projeto
pnpm dev
```
